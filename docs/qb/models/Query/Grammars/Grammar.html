qb.models.Query.Grammars

Class Grammar

lucee.Component
    qb.models.Query.Grammars.Grammar
Direct Known Subclasses:
MSSQLGrammar , MySQLGrammar , OracleGrammar

Grammar represents a platform to run sql on. This is the Base Grammar that other grammars can extend to modify the generated sql for their specific platforms.
